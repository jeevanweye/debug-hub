# Changelog

## [1.0.0] - 2026-01-08

### Added
- Initial release of DebugHub UI
- Floating debug bubble with drag support
- Shake gesture detection to show/hide bubble
- Network monitoring screen with request/response details
- Logs viewer with filtering by level and search
- Crash reports viewer
- Storage browser for app files
- Device and app information screen
- Share functionality for all debug data
- Customizable theme and configuration
- Support for custom additional tabs
- Email sharing with configurable recipients
